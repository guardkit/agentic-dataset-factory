# Data Model: GOAL.md Schema

> Generated by `/system-design` on 2026-03-18

## Entity: GoalConfig

The parsed representation of a domain's GOAL.md file. Central configuration entity consumed by all modules.

## Fields

| Field | Type | Required | Source Section | Description |
|-------|------|----------|---------------|-------------|
| `goal` | `string` | Yes | `## Goal` | What behaviour the fine-tuned model should exhibit |
| `source_documents` | `list[SourceDocument]` | Yes | `## Source Documents` | File patterns with Docling processing modes |
| `system_prompt` | `string` | Yes | `## System Prompt` | System prompt injected into every training example |
| `generation_targets` | `list[GenerationTarget]` | Yes | `## Generation Targets` | Categories, counts, and type split |
| `generation_guidelines` | `string` | Yes | `## Generation Guidelines` | Instructions for the Player agent |
| `evaluation_criteria` | `list[EvaluationCriterion]` | Yes | `## Evaluation Criteria` | Rubric for Coach evaluation |
| `output_schema` | `dict` | Yes | `## Output Schema` | JSON structure for training examples |
| `metadata_schema` | `list[MetadataField]` | Yes | `## Metadata Schema` | Valid values for metadata fields |
| `layer_routing` | `dict[str, str]` | Yes | `## Layer Routing` | Layer → output file mapping |

## Validation Rules

1. All 9 sections must be present — `GoalValidationError` raised if missing
2. `goal` minimum 50 characters
3. `system_prompt` minimum 100 characters
4. `generation_guidelines` minimum 100 characters
5. `source_documents` must have at least 1 entry with valid mode
6. `evaluation_criteria` must have at least 3 criteria with valid Python identifier names
7. `output_schema` must be valid JSON with `messages` and `metadata` keys
8. `metadata_schema` must have all fields marked as required
9. `layer_routing` must contain `behaviour` and `knowledge` entries
10. Generation targets reasoning percentage must be >= 70%

## Invariants

- Evaluation criterion names are valid Python identifiers (used as `criteria_met` keys in Coach rejection)
- System prompt is identical across all generated training examples
- Generation target type distribution matches the reasoning/direct split constraint
