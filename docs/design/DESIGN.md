> Generated by `/system-design` on 2026-03-18

## Purpose

This document captures the detailed design contracts for all six modules in the agentic-dataset-factory pipeline. It resolves the open design questions from the architecture session and defines the schemas, tool contracts, and data flows that ground feature implementation.

## Design Decisions Summary

All 6 open design questions from the conversation starter have been resolved:

| # | Question | Resolution |
|---|----------|-----------|
| DD1 | Full GOAL.md schema | 9 required sections, strict validation at startup (fail fast) |
| DD2 | Coach rejection schema | Extended with `layer_correct`, `type_correct`, structured issues with severity, dynamic `criteria_met` from GOAL.md |
| DD3 | ChromaDB collection naming | Domain directory name (e.g., `gcse-english-tutor`) |
| DD4 | `write_output` layer routing | Validates schema + metadata, routes by `metadata.layer`, append mode, train/val split is post-processing |
| DD5 | Golden set format | `domains/{name}/golden_set.jsonl`, co-located with GOAL.md (optional) |
| DD6 | LangSmith project naming | One project per domain: `adf-{domain_name}` |
